Extensions

  1. Visit the Chrome Web Store to search for an extension.
  2. Copy the URL of the extention to download.
  3. Paste the URL.
  4. Select your Chromium version.
    (auto-detect)

How to Install

  1. Open chrome://extensions. Enable developer mode (top right of screen) and refresh.
  2. Drag-and-drop the CRX from a file browser into the page of the extensions tab.

Notes

  • This is a quick and easy solution to downloading extensions for Chromium-based browsers, HOWEVER:
    For a more intuitive solution with built-in semi-auto-updating, see github.com/NeverDecaf/chromium-web-store.
  • Instructions for downloading extensions manually can be found here.
  • Find your version here: chrome://settings/help
  • Copy + paste the above chrome links in a new tab if your Chromium install doesn't allow opening links to local resources.
