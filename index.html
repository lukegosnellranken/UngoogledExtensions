Extensions

  1. Visit the Chrome Web Store to search for an extension.
  2. Copy the URL of the extension to download.
  3. Paste the URL.
  4. Select your Chromium version.
    (auto-detect)

How to Install

  1. Open chrome://extensions. Enable developer mode (top right of screen) and refresh.
  2. Drag-and-drop the CRX from a file browser into the page of the extensions tab.

Troubleshooting

  • If your browser does not allow you to download, click the copy button instead and paste the URL in a different browser
  • Alternatively, go to chrome://flags and set "Handling of extension MIME type requests" to "Download as regular file"
  • Copy + paste chrome:// links in a new tab if your Chromium install does not allow opening links to local resources.

Adblock for YouTube

  • In recent times, Google has made it increasingly tricky to get around their anti-adblock measures.
  • I've found this extension to be the most reliable, as it skips ads near-immediately as opposed to outright blocking them.
  • Adblock on YouTube is broken as of Chromium v133+. Luckily, the fix is simple. Go to chrome://flags and enable "Disable WebGL." Voilà!
